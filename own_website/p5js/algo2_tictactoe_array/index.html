Fairytale Arrays

This sketch builds on an algorithm which contains an array [-] filled with four other arrays [[1, [2], [3], [4]]. Arrays 1 to 4 contain each 9 strings of nouns from well-known fairytales.

When the refresh button of the browser is clicked the algorithm picks one of the 4 fairytales randomly. At the same time the color for each word is also picked randomly.

The gradient background is also generated with code. Please find it in the related GitHub repository .

You can find the algorithms and website code on my GitHub site.
